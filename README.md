# HR Employee Onboarding Automation (UiPath)

## Project Overview
This project automates the HR employee onboarding process using UiPath (Modern Design).  
It eliminates manual repetitive tasks and ensures faster, error-free onboarding.

## Tools & Technologies
- UiPath Studio (Modern Design)
- Excel Automation
- File System Automation
- Business Rule Handling

## Workflow
1. Read employee details from Excel
2. Validate employee data
3. Create employee folder structure
4. Generate email ID if missing
5. Log welcome email action
6. Update onboarding status in master Excel

## Key Features
- Handles missing/invalid data gracefully
- Uses DataTable processing for performance
- Business-ready automation logic

## Impact
Automated HR onboarding process, reducing manual effort and improving onboarding efficiency.

## Files
- Main.xaml – Main workflow
- Employee_Input.xlsx – Sample input file (dummy data)
